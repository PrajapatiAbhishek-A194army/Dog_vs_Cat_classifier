# Cat vs Dog Image Classifier

A deep learning-based image classification application that predicts whether an uploaded image is a cat or a dog using a Convolutional Neural Network (CNN). The project includes image preprocessing, model training, and deployment through a Streamlit web application for real-time predictions.

## Features

* Cat vs Dog image classification
* Image preprocessing and normalization
* CNN model built using TensorFlow/Keras
* Real-time prediction with Streamlit UI
* User-friendly web interface

## Tech Stack

Python, TensorFlow, Keras, OpenCV, Streamlit, NumPy, Pandas, Matplotlib, [Scikit-learn](https://scikit-learn.org?utm_source=chatgpt.com)

## Usage

1. Upload a cat or dog image
2. Model processes the image
3. Application predicts the class with confidence score

## Outcome

Successfully developed and deployed a CNN-based image classifier capable of identifying cat and dog images with high accuracy.
